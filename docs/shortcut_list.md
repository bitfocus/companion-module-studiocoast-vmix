
# vMix Shortcut List


## Description

This is a list of vMix Shortcuts and the appropriate Companion Action that utilizes them. Whilst all vMix Shortcuts can be run manually through the `Scripting - Run custom command` Action,
the Actions designed for specific shortcuts can include validation, and more appropriately warn when there are issues, to help prevent sending malformed messages to vMix.
